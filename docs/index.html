📱 VirtualPhone

ASP.NET Core

Telegram ↔ Twilio SMS bridge

An ASP.NET Core Web API that forwards messages between your Telegram account and a Twilio SMS number – in both directions.

Telegram user ──► Telegram Bot webhook ──► TelegramController ──► Twilio SMS (outbound) │ Twilio number ◄── SMS reply ◄──────────────────────────────────────────┘ │ └──► Twilio webhook ──► SmsController ──► Telegram chat (forwarded message)

Prerequisites

🔷

.NET 10 SDK

dotnet.microsoft.com/download

📞

Twilio account

A phone number that supports SMS. twilio.com

🤖

Telegram bot

Created via @BotFather

🌐

Public HTTPS URL

Required for webhooks – use ngrok for local dev

Quick start

Clone the repository and fill in your credentials in VirtualPhone/appsettings.json, then:

cd VirtualPhone
dotnet run

Webhook registration

Twilio: In the Twilio Console → Phone Numbers → your number → Messaging, set the incoming webhook URL to:

https://<your-host>/sms/incoming

Telegram: Register the bot webhook with a single curl call:

curl -X POST "https://api.telegram.org/bot<YOUR_BOT_TOKEN>/setWebhook" \
     -d "url=https://<your-host>/telegram/webhook"

Configuration reference

Twilio:AccountSid

Account SID from the Twilio Console

Twilio:AuthToken

Auth Token from the Twilio Console

Twilio:FromNumber

Your Twilio number in E.164 format

Twilio:ToNumber

Destination number for forwarded SMS

Telegram:BotToken

Token provided by @BotFather

Telegram:ChatId

Numeric ID of the target Telegram chat
